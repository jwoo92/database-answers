Here's a Glossary of Useful Terms. and lower down the Page, you will find a list of the Tags and Web Links.

GLOSSARY OF TERMS
TERM DEFINITION
Blog A Blog (short for Web Log) is an online Log or diary, often written by an amateur, discussing anything they feel like talking about. This is often personal stuff but can also be serious thoughts by professionals about their field of expertise. See Wikipedia
Career A Career is a series of Roles that you play over a period of many years of gainful employment. It is good to have a Plan for your career and occupy Roles that show increasing responsibility, either technical or managerial. It is common to find a Role that suits you and then you stay for an extended period or maybe forever. This can be a technically-oriented Role like Database Adminstrator, or a people-oriented Role, like Project Manager or Business Analyst.
Framework Diagram The Database Answers definition of a 6-Stage comprehensive Framework for Data Management.
Global Name Recognition IBM has developped an approach to the important MDM topic of Custmers Matching.
MDM MDM is Master Data Management, which is a very important topic in Enterprise Data Management.
It can be summarised as producing 'A Single View of the Truth'.
Pedagogy The word Pedagogy comes from the Greek and literally means 'To lead the Child'. See Wikipedia
Personalised Learning Personalised_Learning is the tailoring of Pedagogy to individual learners. IT can be a powerful tool because the learning process can be tailored to the individual and progress can be tracked easily. See Wikipedia
Questions Questions and Answers are available with Links to the appropriate Stages in the Framework.
Semantic Web The Semantic Web is a view of the Web where the meaning (semantics) of information and services is defined. This makes it possible for the web to "understand" the requests for information and is Sir Tim Berners-Lee's vision of the Web as a universal medium for data, information, and knowledge exchange. See Wikipedia
Topics These are Topics of Interest related to Databases that people talk about but sometimes have only a vague idea what they mean. For example, MDM and SOA.
Tutorials The Database Answers has produced a wide variety of Tutorials.
User Interface The User Interface has many implications.
User Scenario A User Scenario is a very powerful technique for achieving User involvement and acceptance.
