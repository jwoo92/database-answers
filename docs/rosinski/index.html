  Online Support for Coaching Across Cultures
 

This is a list of facilities which have been developed so far. to support the excellent work that Philippe Rosinski is doing in Coaching Across Cultures.
The approach that we adopted is to identify where the 'Self-Service' principle applies, either in remote Self-Assessment in getting started, or Coaching or Feedback or Tutorials.
This Page is a starting-point for the Online Support for Philippe Rosinski's work.
Here is a Bulletin Board which is a facility for people following Philippe's teaching to share ideas, insights, experiences and so on.
