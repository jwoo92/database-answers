Introduction to Portals
 

Since I first created this page, things have changed in the Portals area. I updated it on July 4th. 2003, to reflect these changes.

Information Portals will become a very important feature in future Internet developments.
A definitive Merrill Lynch Report in 1998 heralded Portals as a 'Killer Application' for Corporate Information Systems.
Since then, a significant number of Portal Sofware products have been produced, some of them from major players, such as IBM and Oracle, and some of them from niche players, such as Hummingbird.

You can learn how to design and build an Information Portal.

Oracle has released a Portal Studio which makes it possible to develop Portals online.

If you would like to be kept up-to-date on Portal developments, please email me.
