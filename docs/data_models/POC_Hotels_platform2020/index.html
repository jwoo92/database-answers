3) Our Business Conceptual Data Model
This is a high-level Model to obtain agreement at the top level.

CRUD Services will include the following :-
1) 'Make a Booking' by Creating a record in the Hotel Events Entity in the Logical Model.
2) 'Check in' by Updating a Booking record or Creating a new record for a Guest who does not have a booking.
3) 'Check out' by Updating the record for a Guest who is checked in.
4) 'Make a Payment' by Creating a record in the Hotel Events Table

4) Physical Data Model
The 'Hotel Users' Entity allows for Users to maintain their own confidential Data.
This Model is used to generate SQL Scripts to create Databases.
