Rules Engine have different degrees of complexity.

In this Basic Level 1 Rules Engine Rules simply have a condition and an action.

The nice thing about this is that many problems naturally fit this model !!!
Here are some typical examples -

  • If the Sun is shining, play Football.

  • If it is raining, take an umbrella.

In this Intermediate Level 2 Rules Engine, Rules have a complex condition but
only one Action.

  • In a car, check the Engine if the Oil Warning Light is flashing.

In this Advanced Level 3 Rules Engine, Rules can have multiple conditions and
many Actions and apply to Assets with very complex structures.
