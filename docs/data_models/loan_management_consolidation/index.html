   Loan Management Consolidation    
An Access Database is available on demand.



The Requirements as defined by Derek Crain are to consolidate data from an
old Loan Approval System with data in a new Loan Management System.

The Data Model shows the data to support the process of matching Loans, Customers and
Addresses and consolidating the data in the old and new Systems to provide Consolidated Loan records.

The Data Flow Processes involve matching Customers to provide a single Consolidated record for any new Loan Application which might become an Approved Loan.
