The Requirement for is a Database to record Student Work Placements.

A. The Things of Interest include :-
A.1 Assignments
A.2 Organizations
A.3 Placements
A.4 Staff
A.5 Students

B. How are they related ?
B.1 Students can be given zero, one or many Placements.
B.2 A Placement is with one, and only one, Organization.
B.3 A Student can be given more than one Placement with the same Organization.
     For this reason, the Start Date of an Placement is part of the Primary Key.
     If this Rule does not apply, then the Start Date is not part of the Primary Key.
B.4 A Placement has an agreed start and end date.
B.5 During a Placement, Students can receive one or more Assignments.
B.6 Each Assignments has a start and end date.
B.7 During an Assignment, a Supervisor will be appointed for the Student.
B.8 At the end of an Assignment, the Supervisor may make Comments on the Student's performance, character, and so on.
B.9 At the end of an Assignment, the Student may make Comments on the Assignment.
B.10 At the end of a Placement, the Organization may make Comments on the Student's performance, character, and so on.
B.11 At the end of an Placement, the Student may make Comments on the Placement.
