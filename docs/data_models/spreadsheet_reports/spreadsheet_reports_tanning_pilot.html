SPREADSHEET REPORTS TANNING PILOT

This diagram shows that data from Sales Spreadsheets is loaded into the Generic Facts Spreadsheet.
The mapping from specific source fields to generic target fields is defined in the table called 'Mapping_Specs_from_User'.
The Report_Views calculate any required Totals, which are then returned to populate the Monthy Sales Report, following
the mapping in the table called 'Monthly_Specs_to_Monthly_Reports'.
