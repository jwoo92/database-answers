Data Model for a Party Master Index   
An Access Database is available on demand.

If you are new to Data Models, this page of my new Tutorial will help you understand the Data Model.
A perennial problem is that Data Modellers sometimes need to talk about Customers and Parties, but business users and SMEs prefer to talk only about Customers because Party can be a difficult concept to grasp at first.
This means we need an approach that is acceptable to both types of people.
I created this Data Model because the Party Master Index looks like it might offer a neat and simple solution to this perennial problem.
