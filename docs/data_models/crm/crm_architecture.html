CRM Loyalty Architecture


The Architecture includes the following Components :- 
  • Loyalty Cockpit - is a desktop portal that displays customer information and actions to streamline customer interactions.

  • Loyalty Interaction History Server - integrates customer information from all of your customer contact channels.

  • Loyalty Outcome Manager - organizes collected customer information in real time for use in analysis and reporting.

  • Loyalty Warehouse - stores behavioral customer data.

    • There is a wide choice of Data Integration Products and eCommerce Products to implement this Architecture.

    Data Islands include :-

  • Business Transaction Systems,(OLTP)
  • Customer DB
  • Interaction DB
  • Resource DB
  • Data Warehouse
    • CIS DB
    • Products
  • RT Marketing Server/DB
  • Quality Server/DB
  • Sales Force/DB
  • Service Server/DB
  • Scripting Server/DB
  • Support Server/DB

    Presentation Layer includes :-

  • Agent Desktop
  • Management Desktop
  • Personalized Desktop
