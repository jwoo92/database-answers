An Access Database is available.

If you are new to Data Models, this page of my new Tutorial will help you understand the Data Model.
You might find this page interesting on Master Data in the Clouds
This page defines the requirements for a Self-Service facility for obtaining Master Data from a variety of sources.

For example, there might be a need for Master Data for Retail business, such as Colours of Socks, Payment Methods and Shoe Sizes.
These could be derived from a variety of sources such as ANSI, ARTS or the BSI (see below).
This would require appropriate Data Governance, and a Data Catalogue to keep track of the Master Data retrieved by each User.


1) User Senario
This shows how Users can create their own sets of Reference Data
in User-specific Catalogues.
