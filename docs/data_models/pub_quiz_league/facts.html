Design for a database for a Pub Quiz League :-
The Application is a Quiz league for the village pubs.

The Database will store about 10,000 questions. The Questions are split into Categores, (in a Table called 'Ref Categories' for Reference Data).
Typical Categories would be Sports, Music, Travels, and so on.

At weekly intervals, or as required, the System will choose 50 questions at random from the database for each pub in the league.

Then, the following week another 50 questions randomly for each pub but not duplicating previous questions.
Each pub would be receiving a completely different set of questions every week and the Application will keep a record of the questions issued to each Pub.
