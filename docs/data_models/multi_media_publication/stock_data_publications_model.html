A DATA MODEL FOR STOCK DATA PUBLICATIONS

The Business Rules have been drafted for Brian's review as follows :-
  • StockData is received on a Daily basis, and is stored in a Database Table called StockData_dd8daily.
  • Different selections of the StockData are published as Products that Customers can subscribe to.
  • These selections are called Views, and each View has a unique View_ID to identify it.
  • Customers subscribe to these Views and pay Subscriptions.
  • Details of which Customers subscribe to which Views will be stored in the Customers_Subscriptions Table.
