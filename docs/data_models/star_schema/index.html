A Star Schema Dimensional Data Model   
This shows a Star Schema Dimensional Model for a Financial Transaction, where a Dimension does not have a higher-level 'Super Dimension'.
Star and Star Data Models are typically used to support Data Warehouses or Data Marts.
These designs are characterised by a central Table, holding the 'FACTS', with a number of 'DIMENSIONS' tables radiating outwards, just to one level.
An Access Database is available on demand.
The Specifications for the Data Model were provided in a question to Database Answers.

We might also design a Data Mart, which is very useful for providng data for Reports and Business Intelligence.
