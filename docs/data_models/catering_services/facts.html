The Requirement is a Database for a Company providing Catering Services,using the Movie industry as an example.

The Database will need to track :-

  • Production company

  • Production Company address


  • Project name or number (many for each production company)


  • Project contact info,i.e: Name address, email (many for each production company)


  • Production/Project Billing Address


  • Physical location of project


  • Start Date


  • End Date


  • Start time


  • End time


  • Wholesale costs for many food items, I.e.: chips, cookies, gummy bears...etc.


  • Wholesale costs for disposable items, I.e.: cups, napkins, ...ect.


  • Packaged Deals i.e.: "gourmet package" includes certain items from wholesale list whereas a
"Budget Package" would include a different list of items from the wholesale list.
Package price would be marked up from the wholesale costs.


  • Many packages would be offered.


  • Package deals would be based on a per person fee for whatever package client has chosen.

>Client would be aware of package prices only and not any wholesale prices.


  • Client will be charged extra fee for using more than the per person allotment.


  • We must be able to put together a useable list of each package offered so client could see what he is paying for.


  • Travel cost to be added to package price if applicable

>Storage cost to be added to package price if applicable


  • Logistic cost for setup and maintenance  based on difficulty of location i.e.: desert, ocean, or
warehouse added to package cost if applicable


  • Per hour labor base price


  • We would need to be able to track costs and profits
