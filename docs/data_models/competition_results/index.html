The User Requirements have been established which define the conditions that the operational Database must meet.

The purpose of this Data Model is to accomodate both individual sports like Golf or Tennis, as well as team sports like Football.
I already have a Data Model for Chess, which should be incorporated in a Generic Model.

The competitor_1_id and competitor_2_id fields identify which two Competitors are participating in a Competiton.
This applies, of course, whether the Competitors are Teams or individual Players.
