  A Data Platform with Web Services
 





This Data Architecture shows a Consolidated Data Platform which provides a conceptual view of Web Services and a Common Data Model.


It uses an Enterprise Service Bus to support an SOA.

The Web Data Services would include a Customer Hub and a Product Master Catalog to provide Master Data Management (MDM).

Data Virtualization is currently a Hot Topic and corresponds to the top level of our Consolidated Data Platform.

The next Layer up includes the Single View and includes personalized user facilities, such as a Profile of Interests (with Publish and Subscribe), Alerts and so on.
