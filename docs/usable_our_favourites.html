These are our current favourites, which we have assessed by some simple, basic principles :-

  1. Does the Home Page load quickly ?
  2. Is the Site visually pleasing ?
  3. Is the navigation easy ?
  4. If 'Flash-y', does it provide a non-Flash option ?
