There is a growing demand for Data Architects 

There is a growing demand for Data Architects, which is an interesting
and natural progression from DBA, Data Analyst and Data Modeller.

The demand for Data Architects is increasing as the Internet and e-Commerce forces many organisations
to rationalise and integrate their Data Sources.

What is a Data Architecture ?
My definition of a Data Architecture is :-
"A Diagram showing the major Components within an overall function through which Data flows."

A well-designed Data Architecture provides insight into questions concerning the data structures and how they relate.
It should provide a starting-point from which Data Models could be created. It should also establish a foundation
for data planning within an enterprise, which includes providing a framework for the creation of Data Models.
