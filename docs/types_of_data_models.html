  Types of Data Models
 
Discussion about the full range of the types of Data Models that we need occurs from time to time.

This is our contribution to the debate, along with this Page.
Please feel free to Email us and let us know what you think..

This table shows a list, with the BI layer at the top, and then going down to the Operational Data Sources.

  Here is our current draft list of the different Types of Data Models
 

 
 
BI Layer Presents the data required for BI Analysis and Reports.The Model generally looks like the Semantic Data Mode, in the next level.
Semantic Data Model This combines presentation of the data with an indication of the meaning of the data.
Here is Wikipedia's view and their example
Conceptual Data Model A business user view of the data.
Data Marts Presents the restricted sets of data required for specific Report families. They usually have the same Dimensions and Facts structure that we find in Dimensional Models.
Dimensional Models Characterised by Dimensions and Facts.
